interactive_calculator
======================

.. autofunction:: LaTeXCalculator.interactive_calculator

Description
-----------

The interactive calculator allows you to perform mathematical calculations using LaTeX syntax. You can enter expressions, equations, or systems of equations, and the calculator will evaluate them and display the results.

You can enter the command ``latexcalc`` in the command line to start the interactive calculator.

.. note::

   You can follow the description of the calculator to use it effectively.
